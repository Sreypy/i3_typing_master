# i3_typing_master

This is a web-based typing test application built for the IPE midterm project.

## 🔧 How to Run

1. Clone the repo  
   `git clone https://github.com/Sreypy/i3_typing_master.git`

2. Open `index.html` in a web browser, or use **Live Server** in VS Code.

## 📁 Pages

- index.html (landing)
- login.html
- register.html
- typing_test.html
- about.html
